##   static 
- 变量:本文件作用域
- 类变量:声明类的静态成员
	- 类的静态成员函数只能访问类的静态成员(变量/函数)
	- 因为static类函数没有this指针
	- 类变量所有类对象共享
- 静态函数同样是本文件内可见(其他文件不能调用)
- 
- 文件内部的所有函数默认都声明为静态的
